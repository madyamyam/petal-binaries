These are the native libraries used for [Petal](https://codeberg.org/yamlrider/Petal).

This repository builds and archives native libraries to be used by Petal.

To download, click on [Actions](https://github.com/madyamyam/petal-binaries/actions) and select the latest build. There will be a file you can download called petal-binaries.zip if you are logged into GitHub. If you are not logged into GitHub or do not have an account, they can also be download with [Nightly](https://nightly.link)
